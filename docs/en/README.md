### Highlights
- **Lightweight**: Core &lt; 30MB, JSON API only
- **High performance**: Vert.x event loop, 50,000+ QPS
- **Type-safe DB**: jOOQ DSL, no SQL injection
- **Stack**: Java 17+, Vert.x 4.5+, jOOQ 3.19+, Maven 3.8+
